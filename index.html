⚡ ¿Cómo mantener el dashboard actualizado?

Tres opciones según tu nivel de automatización deseado. Te recomendamos la Opción B para empezar.

✅ Opción A — Actualización con tu Excel RECOMENDADO HOY

Cada vez que tengas datos nuevos, comparte el Excel actualizado en este chat. En minutos recibes el dashboard regenerado con los nuevos números, gráficas y KPIs. Cero código, cero configuración.

Ventaja: Inmediato, confiable, sin depender de conexiones externas.
Uso ideal: Reportes mensuales o quincenales a directivos.

🔗 Opción B — Google Sheets en vivo (ya configurado en el código)

El código ya está preparado. Solo necesitas dos pasos:

1. Abre tu Google Sheet → Compartir → "Cualquier persona con el enlace" → Lector
2. En el archivo HTML, busca la línea LIVE_DATA: false y cámbiala a LIVE_DATA: true

Luego, cada vez que alguien abra el dashboard, el navegador lee los datos frescos de tu hoja automáticamente. Tú solo actualizas la hoja y el dashboard refleja los cambios.

Requiere: Hoja pública de solo lectura + abrir el HTML desde un servidor (no desde archivo local).

🏢 Opción C — Despliegue web con actualización automática

Para uso corporativo continuo. Publica el HTML en GitHub Pages o Netlify (gratis), conectado a tu Google Sheet pública.

El dashboard tendrá una URL fija que cualquier directivo puede visitar desde el celular o PC, siempre con datos actualizados.

Pasos: Sube el HTML a GitHub → activa GitHub Pages → comparte el link.
https://tuorg.github.io/ugp-dashboard
